SAVEYOURPET REFERENCE IMAGES
=============================

Dieser Ordner enthält Screenshots von saveyourpet.de die als Reference Images 
für die Bildgenerierung verwendet werden.

BENÖTIGTE DATEI:
- website_screenshot_laptop.png  →  Screenshot vom Vergleichsrechner für LAPTOP-Display

VERWENDUNG:
Wenn eine Slide saveyourpet.de erwähnt, wird dieser Screenshot als Reference Image 
an Nano Banana Pro übergeben. Das Tool platziert den Screenshot auf einem LAPTOP-Display 
im generierten Bild.

BILDKONZEPT:
- Over-the-shoulder Perspektive
- Person schaut auf Laptop mit saveyourpet.de
- Haustier schläft/liegt hinter dem Laptop oder daneben
- Beispiel-Setting: Auf dem Bett, Hund schläft hinter dem Laptop

WARUM LAPTOP STATT HANDY:
- Größerer Screen = Logo besser erkennbar
- Natürlicheres "Recherche"-Setting
- Over-the-shoulder Perspektive zeigt Screen-Inhalt optimal

ANFORDERUNGEN AN DEN SCREENSHOT:
- Zeigt den Vergleichsrechner oder eine andere erkennbare Seite
- Logo "saveyourpet.de" sollte sichtbar sein
- Muss nicht perfekt lesbar sein, nur erkennbar
- PNG Format empfohlen
- Empfohlene Größe: ca. 1440x900px (Laptop Viewport) oder ähnlich
